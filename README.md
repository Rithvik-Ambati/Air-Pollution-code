# Air-Pollution-code
AQI forecasting pipeline on CPCB Pune station data (2024–26). Benchmarks 15 models (XGBoost, LightGBM, LSTM, Prophet, SARIMA++) for daily AQI prediction 3 years ahead. Includes pollutant contribution analysis (PM2.5, PM10, NOx, CO, Ozone) and 24+ plots. Built for a research paper
